# CoolWear App 🆒⌚️

This is the app for CoolWear, allowing you to control your CoolWear device via Bluetooth! 🔌📱

# Android Runtime 📱
```
yarn android
```

# Release Build 📦
```
cd android
./gradlew assembleRelease
```
